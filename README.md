# ❗ Die drei !!! Tracker

Lesetracker für alle Die drei !!! Bücher und Hörspiele.

**Live:** https://mycollecto.github.io/drei-ausrufezeichen-tracker/

## Features
- 📚 **133 Bände** in 3 Reihen
- 🖼️ Echte Cover vom Fandom-Wiki
- ✅ Abhaken mit Konfetti
- 🛒 **Amazon Affiliate** – Buch & Hörspiel Buttons pro Band
- 💾 Fortschritt lokal gespeichert
- 📱 PWA – auf Homescreen installierbar

## Reihen
| Reihe | Bände | Status |
|---|---|---|
| Die drei !!! | 119 | aktiv |
| Die drei !!! SUPER | 4 | abgeschlossen |
| Die drei !!! Städte | 10 | aktiv |

## Setup
1. GitHub Pages: Settings → Pages → Branch: main
2. Cover: Actions → „Cover-Bilder herunterladen" → Run workflow
